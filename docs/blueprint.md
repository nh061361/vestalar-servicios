# **App Name**: Firestore Display

## Core Features:

- Data Retrieval: Fetch data from the 'bdvestalar' collection in Firestore, specifically the document 'usuario'.
- Real-time Updates: Listen for real-time updates in the Firestore collection and reflect them in the app.
- Data Display: Display the 'nombre' field ('jose' in the example) from Firestore in a user-friendly format.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) to evoke a sense of reliability and clarity in the display of data.
- Background color: Light gray (#F2F0F4), nearly the same hue as the primary, desaturated, for minimal distraction.
- Accent color: Muted blue (#5383D3) for interactive elements.
- Body and headline font: 'Inter', a sans-serif font known for its readability and clean, modern appearance.
- Simple, geometric icons to represent different data fields or categories (if applicable).
- Clean and structured layout with clear visual hierarchy to make data easy to understand.
- Subtle animations for data loading and updates to provide visual feedback to the user.
